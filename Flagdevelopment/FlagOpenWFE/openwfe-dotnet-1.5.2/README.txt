README for OpenWFE-dotnet

OpenWFE-dotnet is an access library to the worklist (The OpenWFE system in general).

In order to test OpenWFE, you have to download the latest OpenWFE binary release (currently openwfe-1.1.0.zip),
unpack it, and run the start-rest-openwfe.bat script.

You can find binary releases (and source releases) under :
http://sourceforge.net/project/showfiles.php?group_id=54621&package_id=49295

The test.cs file contains client code to OpenWFE-dotnet. It demonstrates
how to use this libary.

If you have questions and feedback, don't hesitate to use the forums :
http://sourceforge.net/forum/?group_id=54621
